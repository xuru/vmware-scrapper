Data Object - HostInStandbyMode

Extends
VmDeployed

Data Object Description

An agent virtual machine is expected to be removed from a host, but the agent virtual machine has not been removed. The reason is that the host is in standby mode.

This is an active remediable issue. To remediate, vSphere ESX Agent Manager puts the host in standby mode and removes the agent virtual machine.
